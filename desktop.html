Aplikasi Lirik Lagu Beranimasi

Aplikasi ini menampilkan lirik lagu secara bertahap di dalam TextField. Tersedia dua tombol: Start dan Pause.

Pada awalnya, hanya tombol Start yang aktif. Setelah ditekan, lirik mulai muncul, tombol Pause aktif, dan Start nonaktif. Menekan Pause akan menghentikan sementara dan mengubah tombol menjadi Continue. Tekan Continue untuk melanjutkan lirik dari posisi terakhir.

Aplikasi Kasir Sederhana

Proyek ini merupakan sebuah antarmuka pengguna yang memungkinkan pengguna untuk melakukan perhitungan total harga suatu barang berdasarkan input yang diberikan. Terdapat tiga TextField utama untuk menginput: nama barang, harga per unit, dan jumlah barang.

Setelah pengguna mengisi data pada ketiga TextField tersebut, mereka dapat menekan tombol Hitung. Sistem akan secara otomatis menghitung total harga dengan mengalikan harga per unit dengan jumlah barang, kemudian menampilkan hasil perhitungan tersebut pada TextField Total.

Aplikasi Kasir Sederhana (2)

Proyek ini merupakan aplikasi perhitungan harga barang dengan fitur perhitungan diskon otomatis berdasarkan jumlah pembelian. Antarmuka terdiri dari beberapa TextField yang digunakan untuk menginput data: nama barang, harga per unit, dan jumlah barang. Setelah pengguna mengisi semua data tersebut, tombol Hitung dapat ditekan untuk memproses perhitungan secara otomatis.

Sistem akan terlebih dahulu menghitung total harga dengan mengalikan harga satuan dan jumlah barang. Jika jumlah barang 6 atau lebih, maka sistem akan memberikan diskon sebesar 10% dari total harga. Nilai diskon ini akan ditampilkan pada TextField Diskon, dan nilai akhir yang harus dibayar (total dikurangi diskon) akan ditampilkan di TextField Bayar. Sebaliknya, jika jumlah barang kurang dari 6 , maka diskon tidak akan diberikan (bernilai 0), dan nilai di TextField Bayar akan sama dengan total harga.

Aplikasi Jam Digital

Aplikasi ini menampilkan jam, menit, dan detik saat pertama kali dijalankan. Waktu bersifat statis hingga pengguna menekan tombol Start, yang akan memperbarui tampilan ke waktu saat ini dan memperbaruinya secara real-time.

Tombol Stop menghentikan pembaruan waktu, dan menekan Start kembali akan menyegarkan waktu dan melanjutkan proses seperti semula. Proyek ini relevan untuk memahami dasar penggunaan timer dan kontrol waktu dalam antarmuka pengguna.

Aplikasi Pengolah Nilai

Proyek ini merupakan aplikasi penilaian mahasiswa berbasis input data. Pengguna diminta mengisi TextField berupa NIM, Nama, Nilai Absen, Nilai Tugas, Nilai UTS, dan Nilai UAS. Sistem akan menghitung nilai akhir (angka) berdasarkan bobot: Absen 10%, Tugas 10%, UTS 30%, dan UAS 50%.

Setelah perhitungan nilai angka selesai, sistem akan menentukan nilai huruf, nilai bobot, dan keterangan, dengan ketentuan sebagai berikut:

  • >84 → A, Bobot 4, Sangat Memuaskan
  • >70 → B, Bobot 3, Memuaskan
  • >55 → C, Bobot 2, Cukup Memuaskan
  • >39 → D, Bobot 1, Kurang Memuaskan
  • ≤39 → E, Bobot 0, Gagal

Seluruh hasil perhitungan akan langsung ditampilkan pada TextField masing-masing.

Aplikasi Peramal

Aplikasi ini merupakan program ramalan sederhana berdasarkan nama pengguna. Pengguna akan diminta memasukkan nama lengkap ke dalam TextField. Setelah itu, pengguna dapat menekan tombol Ramal untuk melihat hasilnya.

Sistem akan menghitung jumlah karakter dalam nama yang dimasukkan, lalu melakukan operasi modulus 6 terhadap panjang nama tersebut. Nilai sisa bagi ini digunakan sebagai dasar untuk menampilkan hasil ramalan tertentu. Setiap angka (0–5) memiliki pesan ramalan yang berbeda.

Aplikasi Pengonversi Terbilang

Aplikasi ini dirancang untuk menghitung total harga pembelian sekaligus mengonversinya ke dalam bentuk teks (terbilang). Pengguna diminta memasukkan nama barang, harga, dan jumlah barang pada TextField yang tersedia.

Saat tombol Hitung ditekan, sistem akan mengalikan harga dengan jumlah untuk menghasilkan nilai total. Nilai total ini kemudian dikonversi ke dalam bentuk terbilang dalam Bahasa Indonesia dan say dalam Bahasa Inggris.

Semua hasil, yakni total harga, versi terbilang, dan versi bahasa Inggris, akan ditampilkan otomatis pada TextField masing-masing.

Aplikasi Jam Sukat (Stopwatch)

Aplikasi ini merupakan stopwatch digital yang menampilkan jam, menit, detik, dan milidetik. Awalnya, tampilan menunjukkan 0:0:0:1 dengan tombol Start yang aktif dan Reset yang nonaktif.

Saat Start ditekan, stopwatch berjalan dan tombol berubah menjadi Stop. Menekan Stop akan menghentikan waktu, tombol kembali ke Start, dan Reset menjadi aktif. Tombol Start dapat digunakan lagi untuk melanjutkan waktu.

Menekan Reset akan mengatur ulang stopwatch ke 0:0:0:0.
